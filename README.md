# DQVCH — Data Quality Vocabulary for Cultural Heritage

A vocabulary of data quality dimensions for descriptive information about cultural heritage objects, modelled using [W3C DQV](https://www.w3.org/TR/vocab-dqv/).

## Documentation

HTML documentation (generated by [Widoco](https://github.com/dgarijo/Widoco)):  
→ **https://mmarkus1.github.io/dqvch/**

## Vocabulary File

The vocabulary is provided as a Turtle file: [`dqvch.ttl`](dqvch.ttl)

**Namespace:** `https://w3id.org/dqvch#`  
**Prefix:** `dqvch:`

## Publication

This vocabulary accompanies:

> Matoni, M., Kesper, A., & Taentzer, G. (2026). Quality of Descriptive Information on Cultural Heritage Objects: Definition and Empirical Evaluation. *arXiv preprint arXiv:2602.21249*. https://doi.org/10.48550/arXiv.2602.21249
